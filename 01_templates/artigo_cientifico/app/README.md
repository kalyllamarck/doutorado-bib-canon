# Renderizadores do Artigo Científico

## Saída CANÔNICA: `.docx`

Use `docx/render_docx.py`. É o caminho oficial desde 2026-06.

```
app/docx/   ← CANÔNICO — veja docx/README.md
```

## Alternativos / Legado

```
app/latex/  ← LEGADO — funciona, mas não é mais o padrão
app/gdocs/  ← ALTERNATIVO — útil para colaboração online no Drive
```

Os renderizadores LaTeX e Google Docs não serão removidos, mas novas
regras canônicas serão aplicadas primeiro ao `.docx`.
